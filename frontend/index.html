Alt tag란 무엇인가요?

Alt tag(대체 텍스트)는 이미지에 붙이는 설명 글로, 눈으로 이미지를 볼 수 없는 사용자나 검색엔진이 이미지의 의미를 이해할 수 있도록 도와줍니다.

📌 왜 Alt tag가 필요할까요?
- 웹 접근성: 시각장애인 사용자는 스크린 리더(Screen Reader)를 통해 Alt tag를 들으며 이미지를 이해합니다.
- 정보 전달: 이미지가 로딩되지 않더라도 Alt tag가 대신 표시되어 내용을 알 수 있습니다.
- 검색엔진 최적화(SEO): 검색엔진은 Alt tag를 참고해 이미지를 인덱싱하므로 노출 효과가 좋아집니다.
- 사용자 경험 개선: 블로그·쇼핑몰 등에서 이미지 의미를 정확히 전달할 수 있습니다.

하지만 현실적으로 사람이 이미지마다 일일이 Alt tag를 직접 작성하는 것은 번거롭고 시간이 많이 걸립니다. 그래서 많은 사이트에서 Alt tag가 비어 있거나 대충 작성되는 경우가 많습니다.

👉 이 웹앱은 AI를 활용해 이미지를 업로드하면 자동으로 Alt tag를 생성합니다. 이미지 속 텍스트를 읽어내는 OCR과 이미지를 이해하는 멀티모달 AI를 결합한 하이브리드 방식 덕분에, 단순한 이미지 설명이 아니라 실제 장면 + 텍스트까지 반영된 정확하고 풍부한 Alt tag를 제공합니다.

이 웹앱은 어떻게 동작하나요?

사용자가 이미지를 업로드하면 자동으로 Alt tag가 생성됩니다.

동작 과정은 다음과 같습니다:

  1. 텍스트 인식 (OCR) → PaddleOCR (korean_PP-OCRv4_mobile) → 바이두(Baidu) PaddlePaddle 팀 개발 → 이미지 속 한국어/영문 텍스트를 빠르고 정확하게 추출
  2. 이미지 장면 이해 & 설명 생성 → Qwen2.5-VL-7B-Instruct → 알리바바 클라우드(Qwen 팀) 개발 → 이미지 장면과 OCR 텍스트를 결합해 자연스럽고 의미 있는 Alt tag 생성

즉, OCR + 멀티모달 AI 하이브리드 구조를 통해 단순히 이미지를 설명하는 것을 넘어서, 실제 이미지 속 텍스트까지 반영한 더 정확하고 풍부한 Alt tag를 제공합니다.
