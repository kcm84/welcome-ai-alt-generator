Alt tag란 무엇일까요?

Alt tag는 이미지의 대체 텍스트를 의미합니다. 시각장애인이 스크린 리더를 사용할 때 이미지를 대신 설명해주고, 이미지가 로딩되지 않을 경우 텍스트로 내용을 전달합니다.

왜 중요할까요?

  • 웹 접근성: 시각장애인·저시력 사용자도 동일한 경험 제공
  • SEO: 검색 엔진이 이미지를 이해하고 검색 노출에 도움
  • UX: 네트워크 불안정 시에도 이미지 내용을 전달

하지만 Alt tag를 작성하는 일은 종종 번거롭습니다. 이 도구는 이미지를 업로드하면 OCR로 텍스트를 추출하거나, 텍스트가 없을 경우 이미지를 설명하여 Alt tag를 자동으로 생성해줍니다.

사용한 AI 모델

이 프로젝트에서는 알리바바 그룹이 개발한 Qwen2.5-VL-Instruct 모델을 사용합니다. 이 모델은 비전-언어 모델(Vision-Language Model, VLM)로, 이미지를 이해하고 텍스트를 읽는 능력(OCR)과, 이미지의 장면을 설명하는 능력을 동시에 가지고 있습니다. 따라서 사진 속 글자를 추출하거나, 텍스트가 없을 경우 장면을 분석해 자연스러운 설명을 생성할 수 있습니다.

  • 📝 OCR 기능: 이미지 속 글자를 인식하여 추출
  • 🖼️ 이미지 설명: 텍스트가 없을 때 장면을 자연스럽게 설명
  • 🌐 활용성: 웹 접근성과 SEO 개선에 바로 적용 가능

Alt tag 생성 과정에서 이 모델이 이미지를 분석하여 100자 내외의 설명을 자동으로 생성합니다.
