Alt tag란 무엇인가요?

Alt tag(대체 텍스트)는 이미지를 설명하는 문구로, 웹 접근성과 검색엔진 최적화(SEO)에 중요한 역할을 합니다.

하지만 문제는, 사람이 이미지마다 일일이 Alt tag를 직접 작성하는 것은 매우 번거롭고 시간이 많이 든다는 점입니다. 이 웹앱은 그 불편함을 해결하기 위해 자동으로 Alt tag를 생성해 줍니다.

이 웹앱은 어떻게 동작하나요?

사용자가 이미지를 업로드하면 자동으로 Alt tag가 생성됩니다.

동작 과정은 다음과 같습니다:

  1. 텍스트 인식 (OCR) → PaddleOCR (korean_PP-OCRv4_mobile) → 바이두(Baidu) PaddlePaddle 팀 개발 → 이미지 속 한국어/영문 텍스트를 빠르고 정확하게 추출
  2. 이미지 장면 이해 & 설명 생성 → Qwen2.5-VL-7B-Instruct → 알리바바 클라우드(Qwen 팀) 개발 → 이미지 장면과 OCR 텍스트를 결합해 자연스럽고 의미 있는 Alt tag 생성

즉, OCR + 멀티모달 AI 하이브리드 구조를 통해 단순히 이미지를 설명하는 것을 넘어서, 실제 이미지 속 텍스트까지 반영한 더 정확하고 풍부한 Alt tag를 제공합니다.
